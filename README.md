# Team-G_Regression

## Body Fat Prediction Using Multiple Regression
A simple Multiple Regression using an existing data set on predicting a person's body fat based on:
* Age
* Weight in pounds
* Neck in cm
* Abdomen in cm
* Thigh in cm
* Forearm in cm
* Wrist in cm
